Build a professional Hebrew RTL website for the Haredi/Orthodox Jewish community.

Website concept:
A digital marketplace/library for ready-made design templates:
- Kiddush invitations
- Wedding invitations
- Bar Mitzvah invitations
- Event posters
- Yeshiva / Torah event ads
- Fundraising posters
- Shabbat and holiday graphics
- Short video templates for events and social media

Main style:
Elegant, clean, premium, Jewish/Haredi aesthetic.
Use deep navy, cream, gold, burgundy, black-and-gold options.
RTL Hebrew layout.
Modern but respectful, not too flashy.

Main features:
1. Homepage with strong hero section:
   Title: "תבניות עיצוב ווידאו לציבור החרדי"
   Subtitle: "הזמנות, מודעות, קליפים ותבניות מוכנות — משנים טקסט ומורידים"
   CTA buttons:
   "צפה בתבניות"
   "העלה בקשה אישית"

2. Template categories:
   - הזמנות לקידוש

   - מודעות לאירועים
   - מודעות לישיבות
   - תבניות וידאו
   - עיצובים לחגים

3. Template cards:
   Each card should show:
   - Preview image/video placeholder

   - Button: "הורדה"

4. Product page:

   - Details

5. Search and filters:

   User chooses template → changes text in form → sees preview mockup → can download or send request by WhatsApp.

7. Add admin-style section placeholder:
